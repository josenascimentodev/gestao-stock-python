# Sistema de Gestão de Stock

Projeto desenvolvido em Python e SQLite para gestão de stock.

## Tecnologias
- Python
- SQLite

## Funcionalidades
- Cadastro de produtos
- Cadastro de categorias
- Entrada de stock
- Saída de stock
- Consulta de produtos
- Consulta de stock

## Próximas versões
- Interface Web com Flask
- Leitura de código de barras
- Alertas automáticos de stock

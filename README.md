# 📦 Sistema de Gestão de Stock

Sistema web desenvolvido em **Python**, **Flask** e **SQLite** para gestão de stock.

Este projeto foi desenvolvido como projeto de portfólio com o objetivo de praticar desenvolvimento web utilizando Flask, organização em Blueprints e integração com banco de dados SQLite.

---

## 🚀 Funcionalidades

- Cadastro de produtos
- Consulta de produtos
- Atualização de stock
- Remoção de produtos
- Gestão de categorias
- Base de dados SQLite

---

## 🛠 Tecnologias

- Python 3
- Flask
- SQLite
- HTML5
- CSS3
- Jinja2

## ▶️ Como executar

Clone o repositório:

```bash
git clone https://github.com/josenascimentodev/gestao-stock-python.git
```

Entre na pasta:

```bash
cd gestao-stock-python
```

Instale as dependências:

```bash
pip install -r requirements.txt
```

Execute a aplicação:

```bash
python run.py
```

Abra o navegador:

```
http://127.0.0.1:5000
```
